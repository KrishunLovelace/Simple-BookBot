BookBot: Python Text Analysis Tool - Work Instructions & Technical Overview

1. Purpose
The purpose of this project was to design and implement a Python-based text analysis tool ("BookBot") capable of processing large text files and producing statistical reports on word and character usage.
This project also served as hands-on practive for configuring a professional Python development environment, structuring a project from scratch, and working effectively in Visual Studio Code.

2. Project Goals
Through this project, I aimed to:
    - Configure a professional Python development environment on my local machine
    - Build a complete Python project from scratch using best practices
    - Learn and apply professional workflows using Visual Studio Code
    - Practice writing readable, maintainable Python Code
    - Improve comfort with debugging, iteration, and testing

3. Development Environment Setup
Tools Used:
    - Python 3.13.1
    - Visual Studio Code 1.115.0 (Universal)
    - git (for version control)
    - GitHub.com

4. What BookBot Does
BookBot reads a text file (such as a novel) and produces a formatted report containing:
    - Total word count
    - Frequency of individual characters
    - Character counts sorted by frequency
    - Clean, readable console output

The program normalizes input text, filters non-alphabet characters, and handles large files efficiently. 

5. Program Flow
    a. Read text from a file
    b. Split and count words
    c. Count individual character frequencies
    d. Sort character counts by frequency
    e. Print a formatted report to the console
